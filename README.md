# GalacticView

This project is a web application built to show NASA api provided data through a sleek and modern interface. It leverages React for the frontend and later plans include integrating a backend service for managing blog posts, reviews, and other user-generated content.
Another service planned is to integrate an AI assistant to answer space-related questions.

## Prerequisites
- Node.js (>= 16)
- Yarn or npm
- Recommended editor: WebStorm (macOS)

## Used technologies

- React
- TypeScript
- React Bootstrap
- NASA APIs
